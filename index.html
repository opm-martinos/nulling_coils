    CMEG Coil Design¶

    This repository hosts code for design of biplanar coils for nulling DC fields for operation of Optically Pumped Magnetometers (OPMs)

    Dependencies¶

    • bfieldtools

    Installation¶

    To make an editable install, please do:

    $ pip install -e .

    Examples¶

    Please run make_contours.py
